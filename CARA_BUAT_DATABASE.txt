 CARA MEMBUAT DATABASE - Sistem Manajemen Kepegawaian
═══════════════════════════════════════════════════════════════

PILIH SALAH SATU CARA DI BAWAH:

═══════════════════════════════════════════════════════════════
 CARA 1: MENGGUNAKAN SCRIPT INSTALL (PALING MUDAH)
═══════════════════════════════════════════════════════════════

1. Pastikan XAMPP sudah berjalan (Apache dan MySQL aktif)
2. Buka browser
3. Akses: http://localhost/Kepegawaian/database/install.php
4. Tunggu sampai proses instalasi selesai
5. Login dengan username: hrd, password: admin123

═══════════════════════════════════════════════════════════════
 CARA 2: VIA PHPMYADMIN - IMPORT FILE SQL
═══════════════════════════════════════════════════════════════

1. Pastikan XAMPP sudah berjalan (Apache dan MySQL aktif)
2. Buka phpMyAdmin: http://localhost/phpmyadmin
3. Klik tab "Import" di menu atas
4. Klik "Choose File" dan pilih file: database/BUAT_DATABASE_MANUAL.sql
5. Scroll ke bawah, klik tombol "Go" atau "Import"
6. Tunggu sampai proses import selesai
7. Login dengan username: hrd, password: admin123

═══════════════════════════════════════════════════════════════
 CARA 3: VIA PHPMYADMIN - COPY PASTE SQL
═══════════════════════════════════════════════════════════════

1. Pastikan XAMPP sudah berjalan (Apache dan MySQL aktif)
2. Buka phpMyAdmin: http://localhost/phpmyadmin
3. Klik tab "SQL" di menu atas
4. Buka file: database/BUAT_DATABASE_MANUAL.sql
5. Copy seluruh isi file tersebut
6. Paste di kotak SQL di phpMyAdmin
7. Klik tombol "Go"
8. Tunggu sampai proses selesai
9. Login dengan username: hrd, password: admin123

═══════════════════════════════════════════════════════════════
 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════

Error: "Unknown database 'kepegawaian_db'"
→ Database belum dibuat, gunakan salah satu cara di atas

Error: "Access denied for user"
→ Cek password MySQL di file config/database.php

Error: "MySQL service not running"
→ Start MySQL service di XAMPP Control Panel
